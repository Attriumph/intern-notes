# 文件结构

**整体结构**： 小程序包含
* 描述整体程序的 app
* 多个描述各自页面的 page

## 主体部分

 一个小程序主体部分由三个文件组成，必须放在项目的根目录，如下：

|文件	 |必填 |	作用|
|------|------|------|
|app.js	|是	|小程序逻辑
|app.json|	是	|小程序公共设置
|app.wxss|	否	|小程序公共样式表

## 每个页面部分
一个小程序页面由四个文件组成，分别是：

|文件类型|	必填	|作用|
|------|-------|------|
|js	|是	|页面逻辑|
|wxml	|是	|页面结构|
|wxss|	否	|页面样式表|
|json	|否	|页面配置|
