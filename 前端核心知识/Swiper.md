# Swiper快速入门

Swiper用于移动端网站的内容触摸滑动，是纯javascript打造的滑动特效插件，面向手机、平板电脑等移动终端。

## 1.五步使用步骤
 * 下载安装或者使用CDN
 * 添加swiper css和js文件到app or website
 * 在HTML文件中添加swiper HTML layout
 * 在css中设置Swiper元素的size
 * 启用Swiper

ps：[代码案例在官方文档](http://idangero.us/swiper/get-started/)
